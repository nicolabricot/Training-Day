Training Day – EnsiLAN
=============

Ce GITHUB a été conçu pour l'entraînement à la nuit de l'info à l'ENSISA de Mulhouse.
Cette entraînement a eu lieu le samedi 17 novembre 2012 de 9h à 19h au sein de l'école.

L'équipe est composé de cinq élèves de 1ère et 2ème années :

- DEVENET Nicolas
- WODITSCH Karl
- LACROIX Florent
- LESPAGNOL Rémy
- ALLENDER Quentin
- DENAUX Robin


Le sujet était : "Tournoi de Jeux-Vidéo"
Nous avons donc créer un site "EnsiLAN"
Ce site sera le plateau du jeu en LAN de l'Ensisa , chaque personne pourra se créé un compte sur le site, il pourra renseigner 
ses informations le concernant. Ce site accueille dse tournois de jeux-vidéo. Chaque utilisateurs pourra s'insrire pour les 
tournois afin d'y participer.

Nous avons développer ce site en HTML/CSS/PHP et avec l'aide de Photoshop.

Nous avons, durant cet entraînement, réaliser des défis tels que :
-Réalisation d'un diagramme UML
-Gestion des sources avec GITHUB
-Accessibilité
-Référencement
-Etc...
